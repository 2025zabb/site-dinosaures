 Site Dinosaures - Projet HTML/CSS

## Objectif

Mettre en pratique :

* la structure HTML
* le responsive design
* la mise en page avec CSS
* la création de formulaires
* l’interactivité avec JavaScript

---

##  Description

Site web responsive réalisé dans le cadre d'utilisation du HTML/CSS.

Ce projet présente une galerie de dinosaures avec une interface moderne, un formulaire de contact et des fonctionnalités interactives en JavaScript.

---

##  Fonctionnalités

*  Galerie de dinosaures interactive
*  Formulaire de contact
*  Vidéo YouTube intégrée
*  Message de bienvenue en JavaScript
*  Design 100% responsive

---

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
* Google Fonts
